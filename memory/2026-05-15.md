## 2026-05-15 — Гопник-раннер: фиксы коллизии и прыжка

### Высота препятствий
- `crate`: 70 → 55px
- `wall`: 90 → 70px
- Длинный прыжок (-600) даёт запас ~86-100px, все препятствия проходимы.

### Фикс коллизии
- **Было**: `player.x/y` вместо `body.x/y` давало смещение ~20px. Коллизия считалась по физическому телу с offset.
- **Стало**: визуальные границы спрайта (`displayWidth/displayHeight`) с небольшим padding.
- Низкие препятствия (box 25px, barrel 35px) теперь убивают если не прыгать.

### Полная переработка прыжка
- **Было**: короткий прыжок на `pointerdown`, длинный на `pointerup` при `isGrounded && !longJumpFired` — баг: длинный срабатывал при посадке, когда `isGrounded` уже ложный.
- **Стало**: прыжок на **отпускание** (`pointerup`). Короткий тап (<150 мс) = короткий прыжок, задержка (>150 мс) = длинный. Порог уменьшен с 200 до 150 мс.
- Убран `longJumpFired` — больше не нужен.
- `tapThreshold` в `config.js`: 200 → 150.

### Метод отладки через инжекцию
- Для тестирования инжектировал `window._deathLog` через `scene.checkCollisions` — логировал тип препятствия, playerBottom, obsTop, расстояние при смерти.
- Это работало через `page.evaluate` в открытой вкладке.


## 2026-04-25 — СтройКонтроль AI v2.1

### Создана архитектура проекта
- `stroikontrol-ai/` — корневой проект
  - `backend/` — FastAPI, SQLAlchemy, SQLite (dev) / PostgreSQL (prod)
  - `mobile/` — React Native + Expo (не запущен из-за Node 24)
  - `ai-models/` — YOLOv8 CV + Audio CNN (не собран в Docker)
  - `frontend/` — React + Vite экспертный дашборд ✅ собран
  - `infra/` — Docker, Nginx, K8s configs
  - `docs/` — API docs, архитектура

### Backend (FastAPI)
- SQLite для dev (aiosqlite), PostgreSQL для prod (asyncpg)
- Модели: users, projects, photos, audio_samples, defects, payments, expert_review_queue
- API endpoints: auth (SMS OTP), projects, analysis, reports, payments, experts
- Запущен на `http://localhost:8001` (uvicorn) и `http://localhost:8002` (Docker)
- Swagger docs: `/docs`

### Docker Compose
- ✅ `api` — FastAPI на порту 8002
- ✅ `postgres` — PostgreSQL 15 на порту 5432
- ✅ `redis` — Redis 7 на порту 6379
- ✅ `minio` — S3-compatible на порту 9000/9001
- ✅ `dashboard` — React экспертный дашборд на порту 3001
- ✅ `nginx` — reverse proxy на порту 80
- ❌ `ml-worker` — убран (CUDA image недоступен + torch 755MB timeout)

### Экспертный дашборд (frontend)
- React 18 + TypeScript + Vite
- Zustand для auth store
- Страницы: Login (SMS OTP), Queue (список задач), Review (просмотр дефектов + вердикт)
- Собран в Docker (multi-stage: node builder → nginx)
- Доступен на `http://localhost:3001`

### Windows-специфичные фиксы
- `psycopg2-binary` → `aiosqlite` (dev)
- `JSONB` → `JSON` в моделях (SQLite)
- `ARRAY(String)` → `JSON` в моделях (SQLite)
- `PostgresDsn` → `str` в Pydantic config
- Порт 8000 занят → используем 8001/8002
- Nginx: убраны ссылки на удалённые сервисы (dashboard)
- SSL pip errors в Docker → `--trusted-host pypi.org`

### Что не работает
- Mobile (Expo) — Node.js 24 несовместим с Expo SDK 50. Требуется Node 18 LTS.
- ML worker в Docker — требует GPU или огромный torch download. Для dev запускать локально через Python.

### Команды запуска
```powershell
# Backend (dev, напрямую)
cd stroikontrol-ai/backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8001

# Docker (все сервисы)
cd stroikontrol-ai
docker-compose up -d

# Frontend dev (если Node 18 установлен)
cd stroikontrol-ai/frontend
npm install
npm run dev
```

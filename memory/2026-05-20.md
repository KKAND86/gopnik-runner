## 2026-05-20 — Выравнивание персонажа относительно земли (поздний вечер)

### Проблема
Персонаж визуально проваливался ниже зелёной линии земли. Пользователь неоднократно указывал, что ноги ниже уровня травы.

### Причина
Phaser Arcade Physics `body.setSize()` с неявным автоцентрированием работает некорректно, когда texture size (12×20 пикселей) отличается от display size (48×64 пикселей). Phaser вычисляет offset относительно texture, а не display, что смещает физический хитбокс относительно визуального спрайта на ~22 пикселя вниз.

### Неудачные попытки
1. `body.setSize(w, h * 0.8)` — тело меньше спрайта, ноги проваливаются
2. `body.setSize(w, h, false) + setOffset(...)` — offset вычислен неправильно, спрайт парит над землёй
3. `body.setSize(w, h, true)` с body < display — Phaser плохо считает offset, спрайт парит

### Итоговый фикс
```js
// Player.js — body ровно равен display size, offset = 0
this.body.setSize(GAME.player.width, GAME.player.height, true);
```
Это единственный вариант, при котором физический хитбокс и визуальный спрайт совпадают идеально.

### Те же изменения применены к другим сущностям
- **Gopnik** — `body.setSize(32, 50, true)`
- **Obstacle** — `body.setSize(w * 0.7, h * 0.7, true)` (не идеально, но лучше)
- **Booster** — `body.setSize(28, 28, true)`

### Source
- `gopnik-runner/src/entities/Player.js` — setSize matches displaySize
- `gopnik-runner/src/entities/Gopnik.js` — setSize with true
- `gopnik-runner/src/entities/Obstacle.js` — setSize with true
- `gopnik-runner/src/entities/Booster.js` — setSize with true

## 2026-05-20 — Починка звука (поздний вечер)

Web Audio API требует user gesture для активации AudioContext. Браузер блокирует звук до первого клика пользователя.
